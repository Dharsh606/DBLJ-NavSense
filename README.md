# AR_NAV_VI - DBLJ NavSense

🧭 **Sense the Path. Navigate the World.**

A comprehensive navigation assistant designed specifically for visually impaired users, featuring AI-powered guidance, Bluetooth device control, and emergency assistance capabilities.

## 🌟 Features

### 🎯 Navigation Mode
- **Voice-Guided Navigation**: Real-time turn-by-turn directions with Text-to-Speech
- **Location Tracking**: GPS-based positioning and route guidance
- **Intelligent Route Planning**: Optimized paths for safe navigation
- **Visual Status Display**: Clear, high-contrast interface for accessibility

### 📡 Surround Scan
- **Obstacle Detection**: AI-powered scanning of surrounding environment
- **Safe Path Analysis**: Identifies clear routes and potential hazards
- **Voice Alerts**: Audio feedback on detected obstacles
- **360-Degree Awareness**: Comprehensive environmental scanning

### 🔗 Bluetooth Device Control
- **Device Management**: Connect and control Bluetooth devices
- **Volume Control**: Adjust audio settings for optimal hearing
- **GPS Integration**: Manage location services
- **Voice Status Updates**: Spoken device status information

### 🚨 Emergency Dashboard
- **SOS Emergency**: One-tap emergency alert system
- **Medical Alerts**: Quick access to medical information
- **Location Sharing**: Share real-time location with emergency contacts
- **Voice Assistance**: Hands-free emergency help

## 🎨 Design Philosophy

- **Accessibility First**: High-contrast colors, large text, and voice guidance
- **Modern UI**: Beautiful gradient backgrounds and card-based layouts
- **Intuitive Navigation**: Simple, gesture-based controls
- **Professional Design**: Clean, modern aesthetic with thoughtful micro-interactions

## 🎯 Target Audience

Designed specifically for:
- Visually impaired individuals
- Users with mobility challenges
- Anyone needing enhanced navigation assistance
- Caregivers and support professionals
